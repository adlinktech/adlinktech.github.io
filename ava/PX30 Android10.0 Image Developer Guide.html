Verify md5sum

After successfully downloaded, we need to use md5sum to make sure that no package is been lost. Example “md5sum PX30_Android.tar.gz.segmenta*”

Merging

We need to merge all the downloaded segments together and name as “ PX30_Android.tar.gz “.

Image Flashing

Go to the directory IMAGE/PX30_ANDROID10_USERDEBUG__20201221.1947/IMAGES/update.img to get the image we need to flash. The image to be flash is the update.img.

Flashing tool

We need to use the SD_Firmware_Tool.exe to do the flashing, but the SD disk tool can only be executed in Windows, please use windows OS for SD card image preparation. After flashing the update.img you can insert the bootable SD card into the target board and boot it.
